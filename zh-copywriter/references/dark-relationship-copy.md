# Adult Dark Relationship-Game Copy

Use this reference when the user provides Chinese relationship, dating, attraction, Alpha/Beta, frame control, emotional push-pull, shit test, cold read, anchoring, scarcity, "上头", "拿捏", "驯化", "恋爱脑", "女性思维", or fan-group conversion material and wants an adult, dark, pressure-heavy Gary-style long script.

Default task: expand a short draft or rough idea into a 4800-5600 Chinese-character long-form oral script. If the user specifies another length, obey it. If the user asks for a short video draft, compress to 1000-1500 Chinese characters.

Do not stop at a concise complete draft. The sample-expanded style is intentionally dense, overexplained, pressure-heavy, and repetitive in an oral-script way. A good output should feel like an internal uncut long video script, not a polished short essay.

## Core Positioning

Do not turn this substyle into positive-growth writing, warm relationship advice, healing content, or mainstream healthy-relationship coaching.

The goal is not to comfort the reader. The goal is to make the target male reader feel exposed, judged, awakened, and pulled into a cold system of human nature, power, frame, and relationship games.

Write with:
- Adult relationship-game framing
- Cold human-nature analysis
- High/low position reversal
- Power and frame control
- Emotional manipulation recognition and counterplay
- Strong/weak, hunter/prey, provider/controller contrasts
- Jungle-law pressure

Avoid these default directions:
- 成为更好的自己
- 双向奔赴

- 值得被爱

Do not neutralize sharp words automatically. Preserve the user's desired dark tone when they explicitly ask for it.

## 5000-Character Expansion Architecture

Build long scripts with this approximate structure:

1. Opening filter, 300-500 characters  
   Start hard. Signal that this is a complete, uncut, adult, high-pressure breakdown, not for naive beginners. Tell the reader why this piece matters now. Mention saving, repeated listening, or asking a summarization assistant for a mind map when useful.

2. Pain-point judgment, 500-700 characters  
   Name the low-status behavior. Shame the common mistake. Make the reader feel that his current strategy is making him prey.

3. Dark mechanism thesis, 600-800 characters  
   Raise the issue from a surface behavior to human nature, mating instinct, subconscious evaluation, power structure, or frame control.

4. Tactical breakdown, 3000 characters  
   Usually split into 3-6 core points. Each point should be a full tactical unit of 900-1300 Chinese characters when the source supports it:
   - Core judgment
   - Brutal judgment
   - Low-level male mistake
   - Specific mini-case
   - Female subconscious interpretation
   - Psychology / biology / behavior mechanism
   - High-level counterplay
   - One dark summary sentence

5. Power summary, 500-700 characters  
   Rebuild the binary contrast: weak/strong, prey/hunter, provider/controller, judged/rule-maker.

6. Core fan-group CTA, 300-500 characters  
   Say public content can only cover the skeleton. Put live cases, chat records, timing, and tactical details in the core fan group.

If the draft has only 3 core methods, each method should be expanded deeply rather than adding many shallow points. The target shape is usually: 1 heavy opening + 1 mechanism thesis + 3 long tactical blocks + 1 power summary + 1 long CTA.
