# Platform Patterns

## Xiaohongshu

Use a personal, concrete, high-signal opening. Prefer lived experience, contrast, checklist, before/after, and "I wish I knew this earlier" framing. Keep paragraphs short. Titles can use curiosity, pain point, result, or identity cues, but avoid fake medical/financial certainty and exaggerated absolute claims.

Useful shapes:
- Pain point -> personal discovery -> 3-5 lessons -> gentle CTA
- Mistake list -> correction -> example -> save/share prompt
- Scenario hook -> concise method -> concrete result -> who it is for

## WeChat Official Account

Use stronger structure and more complete argumentation. Start with a scene, question, contradiction, or observation. Build with subheadings. End with a memorable summary or shareable sentence. Avoid too many emojis or fragmented one-line paragraphs unless the brand voice requires it.

Useful shapes:
- Observation -> why it matters -> deeper cause -> method -> conclusion
- Story -> insight -> framework -> action list
- Problem -> misconception -> new model -> example -> takeaway

## Douyin / Video Script

Write for spoken rhythm. The first 3 seconds must be clear and concrete. Use short sentences, frequent turns, and visible scene/action cues only when useful. Avoid long subordinate clauses.

Useful shapes:
- "很多人以为 X，其实真正关键的是 Y"
- "如果你正在 X，先别急着 Y，试试这 3 步"
- Hook -> conflict -> method -> example -> CTA

## Zhihu

Lead with a direct answer, then justify. Use precise distinctions, examples, and a calm knowledgeable tone. Avoid obvious marketing language. Good answers often contrast naive views with a more nuanced model.

## Moments / Private Domain

Sound like a real person, not a poster. Prefer story, observation, small win, behind-the-scenes, or customer insight. Use a light CTA. Avoid dense slogans and over-designed formatting.

## Landing Page / Sales Page

Prioritize conversion clarity:
- Who it is for
- Pain or desired outcome
- Specific promise
- Mechanism or differentiator
- Proof
- Objection handling
- Clear CTA

Keep claims verifiable. Replace "best", "guaranteed", and "秒变" with specific outcomes and conditions.
